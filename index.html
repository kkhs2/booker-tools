Customer File creator

An attempt to make this quicker / easier? A few points:

  • Red asterisk indicates required fields
  • Check GitHub to find the last account number created in order to not overwrite any current accounts
  • Accounts created should have the format of CustomerID@test.com
  • Only required fields are showing, non-required fields are created with spaces in the background, or with default values (such as addresses)
  • "Try" to create an option to create a combination of accounts with one hit of a button. The combination is based on Branch ID, Group Sector (which controls whether it is a Retailer/Caterer), Licenced
  • An option to add the values selected as a new line
  • An option to add the values selected and generate a useful combination of lines, based on business type, group sectors and tobacco club
